开发自己的小部件

本指南不是HTML，CSS和JavaScript教程。如果您想开发自己的小部件，则希望您至少了解这些前端技术的工作原理。一个开始学习的好地方是在w3schools.com上或下载SoloLearn应用程序。

•找到“Widgets-Folder”

此文件夹应在C://Users/MY_USER/Documents/Widgets下

•创建您的HTML小部件

创建一个html文件并根据需要代码（html，html， CSS和JS都是有效的）。使用您喜欢的浏览器开发 您的小部件。

•附加小部件窗口的元标记

在您的HTML文档的顶部添加以下标签（仅 需要前2个）：

<meta name =“ applicationTitle” content =“ widget name”>
<meta name =“ windowsize” content =“ 500 500”>
<meta name =“ previewsize” content =“ 500 500”>
<meta name =“ windowborderradius” content =“ 5”>
<meta name =“ WindowLocation” content =“ 500 500”>
<meta name =“ windowopacity” content =“ 255”>
<meta name =“最大” content =“ true”>

•打开Winwidgets并选择您的小部件

您的小部件可以在小部件列表中找到。使用搜尋欄找到它。

保存小部件数据

首先，将您的小部件放在文件夹中（这是一个必须）。然后创建一个称为的配置文件 config.json

C:\Documents\
└── Widgets
└── Weather
├── widget.html
└── config.json

请注意，一旦小部件完成加载，此功能只会发射一次。将数据保存到对象中，并在小部件继续运行时修改该对象。

功能 onGetConfiguration(数据）

从 CefSharp 对象中调用 PostMessage 函数来保存您的数据。数据必须作为 JSON 对象传递。
CefSharp.PostMessage(JSON.stringify({ language: 'en-US' }))

键盘捕获事件

定义功能 OnnativeKeyEvents(鍵） 在您的前端 JavaScript 代碼中接收來自您計算機的全局鍵盤按鍵事件。

此函數將接收一個 JSON 回應。這個回應的屬性會根據外設的不同而有所變化。

// 當事件來自鍵盤時
{
eventType: "keyboardEvent",
state: "keyDown",
keyCode: 214,
}
// 當事件來自鼠標時
{
eventType: "mouseEvent",
xPosition: 80,
yPosition: 214,
buttonPressed: "Right",
buttonPressCount: 12,
mouseWheelOffset: 5
}
